\subsection{Gestion du focus}

Afin d'améliorer l'ergonomie d'utilisation d'\obm, certaines facilités sont disponibles pour la gestion du focus.\\

Le focus est positionné par défaut sur le bouton de soumission du formulaire nommé \variable{f\_search} si existant dans la page.
Il suffit donc d'appeler le formulaire de recherche d'un module \variable{f\_search} et son bouton de validation \variable{submit} pour qu'à l'arrivée dans ce module la pression de la touche ENTREE lance une recherche.
