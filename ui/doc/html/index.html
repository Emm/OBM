About OBM

OBM

O.B.M. stands for Open Business Management.

OBM is an Intranet application which goal is to help manage a company, but can be used as a contact database. It is written by an evolving group of developpers and lead by Pierre Baudracco, with contributions by other people (see the changelog) and released under the terms of the GPL.

This version of OBM is a developper only release as it should be difficult to install and the data and the database model are subject to change.

Requirements

OBM is developped under Linux, Apache, PHP, PHPlib, MySQL or PostgreSQL.

OBM has been tested on these versions :

Software Version Comment
Linux 2.0.31 - 2.4.19 OBM shouldn't be related to kernel and OS version directly
Apache 1.3.3 - 1.3.26 Not tested on 1.2 but should work with
PHP OBM 0.3, 0.4 : 3.0.5 - 3.0.16; 4.0.1 pl2 - 4.1.x
OBM 0.5.x : 4.0.x - 4.2.x (PHP3 no more supported)		 One or two ";" to delete with 4.0.x (for OBM 0.3) but that's all
PHPlib 7 - 7.2 Is included (but modified) in OBM
MySQL 3.22.8 - 3.22.32; 3.23.x  
Postgres 7.1.x, 7.2.x Is supported since 0.3.0
(Version prior to 7.1 (6.5.3, 7.0.x) were supported till OBM 0.5.1)
(Postgres creation scripts are lacking for now)

There are reports of OBM running on :

Platform Reporter Comment
NT with Apache Raymond Gardner  
FreeBSD 4.1, Apache 1.3.12, Mysql 3.23, PHP 3.0.16. Ryan Fox OBM 0.4

All OBM versions support MySQL, and PostgreSQL is supported since 0.3.0. Multidatabase support is done throught Phplib, so it should be easy now to add support for other databases.

Phplib (PHP Base Library) is used in OBM. Only parts needed by OBM are present in this distribution, but Phplib can be obtain at phplib.netuse.de

  • Apache : www.apache.org
  • PHP : www.php.net
  • MySQL : www.mysql.org
  • PostgreSQL : www.postgresql.org
  • Phplib : phplib.netuse.de
